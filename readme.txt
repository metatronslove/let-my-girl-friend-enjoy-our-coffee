A free, self-hosted donation widget that lets you accept credit card (Stripe), PayPal, and cryptocurrency payments inline – no redirects.

== Description ==

**Let My Girl Friend Enjoy Our Coffee** adds a beautiful floating donation widget to your WordPress site. Your visitors can support you via:

* 🧾 **Credit card** – powered by Stripe Elements (card details entered inside the widget)
* 🟡 **PayPal** – embedded Smart Buttons, no redirect
* 🔗 **Cryptocurrency** – display your wallet address (Quai recommended)

**Key Features**

* **Inline payments** – visitors stay on your site; no external pages.
* **Customizable** – choose button color, icon (emoji/SVG/PNG), position, and margins.
* **Donation tiers** – set predefined amounts or allow custom input.
* **Membership support** – upcoming: recurring subscriptions via Stripe.
* **Mobile‑friendly** – responsive modal works on all devices.
* **Free to use** – no monthly fees; only transaction costs from Stripe/PayPal.

= How to Get Started =

1. Sign up for a free hosting account (e.g., InfinityFree) and install WordPress.
2. Install and activate this plugin.
3. Get a free crypto wallet (Quai Network) and copy your address.
4. (Optional) Create a free Stripe account and copy your publishable and secret keys.
5. (Optional) Create a PayPal business account and generate client ID/secret.
6. Enter the details in the plugin's **Settings** and **Payment Methods** tabs.
7. The widget will appear on your site. Done!

== Installation ==

1. Upload the `let-my-girl-friend-enjoy-our-coffee` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings → Coffee Widget** to configure the button.
4. Go to **Coffee Widget → Payment Methods** and enter your payment details.
5. The widget will automatically appear on your site.

== Frequently Asked Questions ==

= Do I need a Stripe account to accept credit cards? =
Yes, but it's free to sign up. You pay only per transaction.

= Can I use other cryptocurrencies besides Quai? =
Yes, you can enter any wallet address. The network is displayed for user guidance.

= Does the widget redirect visitors to another page? =
No – all payments happen inside the modal. Stripe and PayPal are embedded.

= Is the widget responsive? =
Absolutely. It works on desktops, tablets, and mobile phones.

= Can I customize the look? =
Yes, you can change button color, icon, position, margins, and even add custom CSS/JS.

= How do I set up the GitHub support page? =
The plugin creates a page at `/wp-content/plugins/let-my-girl-friend-enjoy-our-coffee/public/github-support.php`. You can use that URL in your GitHub repository's `FUNDING.yml` under `custom:`.
